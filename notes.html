Development notes

The stack

This project uses React.js, React router, Vite, ESLint.

I'm afraid I didn't get around to TypeScript or Vitest.

Time spent

I blew past the 2 hour mark pretty quickly. This took me two solid evenings, there was a lot to do!

It's worth noting that a large part of it was refamiliarising myself with React and React Router.

My previous job used Vue.js. My current work is heavily Web Components based (bring on React 19!). I'm sure I'd be a lot quicker going forward.

CSS and CSS-in-JS

The demo uses a quickly-put-together collection of base CSS.

Component CSS is just native CSS using adopted stylesheets. I skipped a pre-processor and transpiler to save time. As a result, I'd recommend avoiding older browsers :)

I decided to take the new CSS @scope rule for a spin instead of Tailwind. @scope is not supported in Firefox so will look a little broken as a result (but still works!).

Shadow DOM solves a lot of this in my personal projects, but I still found it a useful learning experience.

I'm happy to use Tailwind or the preferred solution of choice.

I didn't focus too much on the looks (tempting!), more on getting the components working well in the time available.
